# Laboratorio6-Chat---Web

Archivos

index.html - Estructura de la interfaz
style.css - Estilos
chat.js - Lógica de la aplicación

Funcionalidades

Envío de mensajes con Enter
Límite de 140 caracteres por mensaje
Auto-refresh de mensajes cada 3 segundos
El scroll se preserva al recibir nuevos mensajes
Preview automático de imágenes detectadas en el texto
Preview automático de enlaces web con título y descripción
Nombre de usuario persistente entre sesiones

Servidor
Los mensajes se conectan al servidor de la clase en https://chat.joelsiervas.online/messages.
